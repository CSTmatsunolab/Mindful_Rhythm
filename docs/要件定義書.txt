要件定義書（モバイルアプリ版）

プロジェクト名：Mindful Rhythm（マインドフル・リズム）
作成日：2025年10月23日
作成者：〇〇ゼミ開発チーム（短大ゼミ2025）

1. 概要
1.1 背景

「Mindful Rhythm」は、心理学（特に認知行動療法：CBT-I）を応用した、睡眠・感情・行動の相関分析と改善支援を行うモバイルアプリである。
従来の睡眠記録アプリは単なる数値ログにとどまっていたが、本アプリではAIと心理学的理論を融合し、ユーザーの生活リズムを整えるための行動支援を目的とする。

1.2 目的

ユーザーが自分の「睡眠・感情・行動」を再認識することで、
　より健康的な生活リズムを形成する。

睡眠データを分析し、科学的根拠に基づいた個別アドバイスを提示する。

「記録アプリ」ではなく「自己理解を支援するパートナー」として機能する。

1.3 コンセプト

「心と体のリズムを整え、毎日をより充実させるための、あなただけのウェルネスコンシェルジュ。」

2. システム構成
2.1 概要

本システムは、スマートフォン（iOS / Android）向けのモバイルアプリケーションとして開発する。
クラウド接続を前提としないローカル完結型アプリとし、センサー（加速度・マイク等）を活用して睡眠の質を推定する。

2.2 対応プラットフォーム
項目	内容
OS	iOS / Android
フレームワーク	React Native（Expo）または Flutter
データ保存	SQLite / AsyncStorage
グラフ描画	Recharts または Victory
センサー利用	加速度・マイク・照度センサー
デプロイ	App Store / Google Play（学内配布可）
3. 機能要件
3.1 機能一覧
区分	機能名	概要
睡眠管理	睡眠記録	就寝・起床時間を入力または自動検知
睡眠管理	睡眠スコア計算	睡眠時間・深睡眠・中途覚醒からスコア算出（0～100点）
睡眠管理	睡眠環境日記	運動・カフェイン摂取・入浴等の環境記録
睡眠管理	アドバイス提示	AIによる睡眠改善提案（BGM／瞑想／呼吸法）
タスク管理	ToDoリスト	日々のタスクを作成・完了・削除
タスク管理	感情付き完了	タスク完了時に感情（絵文字）を記録
タスク管理	睡眠改善課題	毎日1件、自動で行動課題を提示
CBT-I支援	睡眠日記	不安・夢・感情をテキスト／音声で入力
CBT-I支援	AIインサイト	睡眠・感情・行動の関係を自動分析
UI/UX	キャラクター成長	「スリーピン」キャラが達成度に応じて変化
ビジュアル	ホーム背景変化	睡眠状態に応じて背景（太陽／月／曇り）を変化
4. 非機能要件
項目	内容
起動速度	5秒以内
操作レスポンス	主要操作1秒以内
データ保存	オフライン対応（端末内保存）
セキュリティ	データは暗号化してローカル保存
UIデザイン	ダークトーン＋柔らかい配色（夜間重視）
保守性	コンポーネント単位でモジュール管理
